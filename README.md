# GameSwap MVP (Front)

Front-end do MVP de trocas de jogos.

## Rodar local
```bash
npm install
npm run dev
```

## Variáveis de ambiente (opcional por enquanto)
Crie um `.env.local` na raiz (quando entrar auth / api):
```bash
VITE_API_URL=http://localhost:8000
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

## O que já tem
- Tela **Explorar** com filtros e cards (mock data)
- Arquitetura por feature (`src/features/explore`)
- UI base estilo shadcn (componentes em `src/components/ui`)
